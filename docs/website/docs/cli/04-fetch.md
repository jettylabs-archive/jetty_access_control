# fetch

Fetch metadata using an existing Jetty project

### Usage

`jetty fetch [OPTIONS]`

### Options

| Flag                                | Description                                               |
| ----------------------------------- | --------------------------------------------------------- |
| `-c`, `--connectors` `<CONNECTORS>` | Connectors to collect for                                 |
| `-l`, `--log-level` `<LOG_LEVEL>`   | Specify the log level. Can be debug, info, warn, or error |
| `-h`, `--help`                      | Print help information                                    |
