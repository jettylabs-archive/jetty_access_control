# add

Add connectors to an existing Jetty project via a similar flow to `jetty new`

### Usage

`jetty add [OPTIONS]`

### Options

| Flag                              | Description                                               |
| --------------------------------- | --------------------------------------------------------- |
| `-l`, `--log-level` `<LOG_LEVEL>` | Specify the log level. Can be debug, info, warn, or error |
| `-h`, `--help`                    | Print help information                                    |
